普通数组 ＋ 普通数据

    
    var list = [{
        a: 1, b: 1
    }, {
        a: 2, b: 2
    }, {
        a: 3, b: 3
    }];
Field A Field B
无法感知添加事件和变更事件

ko 数组 ＋ 普通数据

    
    var list = ko.observableArray([{
        a: 1, b: 1
    }, {
        a: 2, b: 2
    }, {
        a: 3, b: 3
    }]);
Field A Field B
只能感知添加事件，但不能感知变更事件

ko 数组 ＋ ko 数据

    
    var list = ko.observableArray(
        ko.mapping.fromJS(
            [{
                a: 1, b: 1
            }, {
                a: 2, b: 2
            }, {
                a: 3, b: 3
            }]
        )()
    );
Field A Field B
既能感知添加事件，还能感知变更事件
